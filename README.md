# Music Sub-genre Popularity Trend Analysis

**Objective:** Explore how sub-genres of music have shifted in popularity over time using Python & SQL.

**Dataset:** Spotify Tracks Dataset (link)

---

## Project Overview
This project will:
- Load a music-track dataset with sub-genre labels.
- Clean and standardise genre/sub-genre categories.
- Store the data in SQL (SQLite or PostgreSQL) for querying.
- Analyse and visualise how the popularity of sub-genres has evolved.
- Provide insights and potentially forecast future trends of music sub-genres.
